Provides write and read access to an flat file store called DataFile. Interface to this functionality is the DataFileService class. This service could create an DatabaseHandler for flat file stores of supported formats.

The classes in this package could be divided in three parts:

  • classes that represents the structure or meta data of an DataFile. This classes are builds a DataFileSchema with DataFileColumns and the factories for these objects.
  • classes that represents the stored data as records,like the DataFileRecord and the RecordValue.
  • the DataFileHandler as the class that capsules the IO operations for the DataFileAccess.
